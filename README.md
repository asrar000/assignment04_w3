# This TaskManager app is a modern,reponsive applicaiton that is built using React and Tailwind that fetches and display tasks from JSONPlaceholder API.

## Features 
(i)Task Management - View and manage 200 tasks from JSONPlaceholder API
(ii)Search Functionality - Filter tasks by title in real-time
(iii)Smart Pagination - Navigate through 10 pages with 20 tasks per page
(iv)Status Toggle - Mark tasks as complete or incomplete
(v)Persistent State - Task statuses saved locally and persist across page reloads
(vi)Light/Dark Mode - Toggle between light and dark themes
(vii)Responsive Design - Works seamlessly on desktop, tablet, and mobile
(viii)Custom 404 Pages - User-friendly error handling for invalid routes


## Starting the app
The following steps includes cloning the project,opening the root directory,installing all the dependcies and finally starting the app
```bash
git clone https://github.com/asrar000/assignment04_w3
cd task-manager
npm install
npm start
```
